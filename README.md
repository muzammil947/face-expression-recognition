# Facial Expression Recognition 😊😠😢

A real-time Facial Expression Recognition System developed using Python, OpenCV, and Deep Learning techniques. This project detects human emotions through webcam/live video streaming and classifies facial expressions accurately in real time.

---

## 🚀 Features

- Real-time face detection
- Emotion recognition using Deep Learning
- Webcam/live video support
- Fast and accurate predictions
- User-friendly interface
- Detects multiple emotions

---

## 😃 Detected Emotions

- Happy 😊
- Sad 😢
- Angry 😠
- Surprise 😲
- Neutral 😐
- Fear 😨

---

## 🛠 Technologies Used

- Python
- OpenCV
- TensorFlow / Keras
- NumPy
- CNN (Convolutional Neural Network)

---

## 📌 Project Workflow

1. Capture live video from webcam
2. Detect human faces using OpenCV
3. Extract facial features
4. Predict facial emotion using trained model
5. Display emotion label with confidence score
